# Cotizador

Ruta en Turboship Web: `/business/[businessSlug]/quote`

El Cotizador permite comparar opciones de envío por costo, tiempo y servicio
para tomar decisiones comerciales más rentables.

Ayuda a responder más rápido a clientes, proteger margen y mejorar la tasa de
conversión en cada operación.
