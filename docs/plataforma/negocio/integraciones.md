# Integraciones

Ruta en Turboship Web: `/business/[businessSlug]/integrations`

Integraciones conecta la operación con canales de venta como Shopify y
WooCommerce para importar órdenes de forma automática.

Esto reduce carga manual, acelera tiempos de procesamiento y mejora la
continuidad entre venta, operación y cumplimiento logístico.
