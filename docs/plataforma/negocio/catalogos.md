# Catálogos

Ruta en Turboship Web: `/business/[businessSlug]/catalogs`

Catálogos permite administrar remitentes, destinatarios y productos para
estandarizar la captura de datos en toda la operación.

Su uso reduce errores manuales, acelera la creación de órdenes y mejora la
calidad de información para cotizar y enviar con precisión.
