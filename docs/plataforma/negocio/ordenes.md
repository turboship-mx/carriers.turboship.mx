# Órdenes

Ruta en Turboship Web: `/business/[businessSlug]/orders`

Órdenes centraliza la creación y seguimiento de pedidos, incluyendo búsqueda,
filtros y vistas por estatus operativo.

Permite acelerar el flujo desde la venta hasta el embarque, con mejor control
de pendientes y menos retrabajo en la operación diaria.
