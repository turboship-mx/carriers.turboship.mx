# Envíos

Ruta en Turboship Web: `/business/[businessSlug]/shipments`

Envíos concentra el monitoreo de guías activas, canceladas y finalizadas para
dar visibilidad de punta a punta al cumplimiento logístico.

Facilita detectar incidencias a tiempo, atender clientes con información clara
y sostener promesas de entrega con mayor consistencia.
