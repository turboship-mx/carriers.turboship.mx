# Configuración

Ruta en Turboship Web: `/admin/settings`

Configuración centraliza los ajustes globales del sistema:
marca, administradores, recargas, Mercado Pago, módulos y banner.

Es la base para gobernar la plataforma con reglas claras y alinear la
operación con objetivos comerciales y de servicio.
