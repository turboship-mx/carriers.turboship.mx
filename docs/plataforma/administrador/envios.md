# Envíos

Ruta en Turboship Web: `/admin/shipments`

Envíos centraliza la operación diaria de guías: filtros, seguimiento,
actualización de estatus y exportación de información.

Su objetivo es dar control de punta a punta para cumplir promesas de
entrega y reaccionar rápido ante incidencias.
