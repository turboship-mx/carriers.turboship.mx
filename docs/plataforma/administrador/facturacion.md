# Facturación

Ruta en Turboship Web: `/admin/billing`

Facturación consolida recargas, movimientos y filtros de consulta para
control administrativo y conciliación financiera.

Permite detectar variaciones a tiempo, validar información fiscal y
respaldar decisiones de cobranza y operación.
