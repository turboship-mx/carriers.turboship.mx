# Negocios

Ruta en Turboship Web: `/admin/businesses`

Negocios concentra el catálogo de cuentas empresariales y sus indicadores
operativos básicos para administración diaria.

## Qué puedes hacer aquí

- Buscar negocios por nombre, sitio web o usuarios asociados.
- Ordenar por nombre (`A-Z`, `Z-A`) o saldo (menor/mayor).
- Navegar por paginación cuando el volumen crece.
- Crear un negocio nuevo (`Crear Negocio`).
- Entrar a módulos del negocio (usuarios, créditos y configuración) desde las
  acciones por fila.
- Revisar saldo visible por cuenta para control rápido de disponibilidad.

## Prerrequisitos

- Usuario con permisos de administrador.
- Acceso a `/admin/businesses`.

## Recorrido de interfaz

1. Acciones superiores: menú adicional y botón `Crear Negocio`.
2. Barra de búsqueda y selector de ordenamiento.
3. Tabla con columnas `Nombre`, `Usuarios`, `Saldo`.
4. Avatares de usuarios por negocio, con acceso rápido para agregar usuarios.
5. Menú de acciones por negocio para navegación administrativa.

## Flujo recomendado de uso

1. Busca el negocio objetivo por nombre o dominio.
2. Ordena por saldo cuando revises riesgo operativo/financiero.
3. Abre la cuenta desde acciones para gestionar usuarios, créditos o ajustes.
4. Si es onboarding, usa `Crear Negocio` y completa alta inicial.

## Buenas prácticas

- Estandariza nombre y sitio web desde el alta para evitar duplicados.
- Revisa saldo antes de escalar problemas de compra de guías.
- Mantén la asignación de usuarios por negocio actualizada para trazabilidad.
