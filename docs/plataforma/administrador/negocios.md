# Negocios

Ruta en Turboship Web: `/admin/businesses`

Negocios concentra la administración de cuentas empresariales:
alta, configuración operativa, acceso a módulos y seguimiento general.

Es el punto clave para escalar clientes sin perder control de servicio,
rentabilidad y cumplimiento operativo.
